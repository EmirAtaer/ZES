
# ZES Karar Destek Sistemi (KDS)

Türkiye’de elektrikli araç şarj istasyonu yatırımlarını veriye dayalı şekilde planlamak için geliştirilmiş bir karar destek platformudur. Zorlu Energy Solutions (ZES) için hazırlanmıştır, ancak demo olarak herkes tarafından incelenebilir.

## Özellikler

- Türkiye haritası üzerinde mevcut ve potansiyel şarj istasyonlarını görselleştirir.
- Her il için pazar fırsatlarını, yatırım önceliklerini ve ZES’in pazar payını analiz eder.
- Grafikler, haritalar ve özet raporlarla kullanıcıya kolay anlaşılır analizler sunar.
- Kullanıcı dostu ve hızlı arayüz.

## Demo Giriş Bilgileri

- Kullanıcı adı: **admin**
- Şifre: **admin**

## Kurulum ve Çalıştırma

1. Depoyu klonlayın:
   ```bash
   git clone <repo-link>
   cd ZES
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Uygulamayı başlatın:
   ```bash
   npm run dev
   ```
4. Tarayıcınızda `http://localhost:3000` adresine gidin ve yukarıdaki bilgilerle giriş yapın.
